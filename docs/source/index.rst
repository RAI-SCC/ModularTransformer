Transformers Made Modular!
==========================

Transformers and attention models provide a wealth of options to adapt them to a variety of problems and situations.
The ``ModularTransformer`` makes it easier to explore this space for manual experimentation or automatic hyper-parameter search.

.. note::
   This project is under active development. For any issues, feature requests, or contributions, visit the project `GitHub page`_
